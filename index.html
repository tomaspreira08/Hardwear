Requisitos para o Visual Studio Code

Hardware:
- Processador de 1,6 GHz ou superior (Esta é a velocidade com que a CPU pode executar instruções. Processadores mais rápidos proporcionam melhor desempenho, e o Visual Studio Code se beneficia de uma velocidade de processamento decente.)
- 1 GB de RAM (RAM suficiente permite uma operação mais suave e o manuseio de bases de código grandes.)
Plataformas:
- Windows 10 e 11 (64 bits)
- macOS com suporte a atualizações de segurança da Apple
- Linux (Debian): Ubuntu Desktop 18.04, Debian 10
- Linux (Red Hat): Red Hat Enterprise Linux 7, CentOS 7, Fedora 35
Não suportado:
- Soluções de virtualização de aplicativos
- Múltiplos usuários simultâneos na mesma máquina
Requisitos adicionais para Linux:
- Versão GLIBCXX 3.4.21 ou posterior
- Versão GLIBC 2.17 ou posterior


Requisitos para o Blender

Hardware:
- Mínimo: CPU quad-core de 64 bits, 8 GB de RAM, tela Full HD 1920×1080, Placa Gráfica de 2 GB VRAM (OpenGL 4.3)
- Recomendado: CPU octa-core de 64 bits, 32 GB de RAM, tela 2560×1440, Placa Gráfica de 8 GB VRAM
Placas Gráficas Suportadas:
- NVIDIA GeForce 400 e mais recentes
- AMD GCN 1ª geração e mais recentes
- Arquitetura Intel Broadwell e mais recentes
Plataformas:
- Windows 8.1, 10 e 11
- macOS 10.15 Intel · 11.0 Apple Silicon
- Distribuições Linux usando glibc 2.28 ou mais recente


CPU, RAM mínimos e resolução da tela: Esses requisitos garantem que o Blender possa ser executado em um nível básico, fornecendo uma base para usuários com hardware padrão.
CPU, RAM, e resolução de tela recomendados: Para usuários que trabalham com modelos 3D complexos, animações e renderização, é recomendada uma CPU mais potente, mais RAM e uma resolução de tela mais alta para um desempenho ideal.
Placas gráficas suportadas: Requisitos específicos de placa gráfica garantem compatibilidade e renderização eficiente.
Plataformas: A compatibilidade com Windows, macOS e distribuições Linux garante ampla acessibilidade.


Requisitos para o Unity 2022.3

Requisitos mínimos:
- Versão do sistema operacional: Windows 7 (SP1+), Windows 10 e Windows 11 (64 bits)
- CPU: Arquitetura X64 com suporte ao conjunto de instruções SSE2
- API Gráfica: GPUs compatíveis com DX10, DX11 e DX12
Requisitos adicionais:
- Drivers oficialmente suportados pelo fabricante de hardware
- Para dispositivos Apple Silicon, é necessário o Rosetta 2


Sistema operacional: Unity suporta Windows 7 (SP1+), Windows 10 e Windows 11 em sistemas de 64 bits, com suporte a um conjunto específico de instruções (SSE2).
API gráfica: O suporte a DirectX 10, 11 e 12 garante compatibilidade com uma variedade de GPUs.
Requisitos adicionais: Drivers de hardware oficialmente suportados garantem estabilidade e desempenho.
Para dispositivos Apple Silicon: É necessário o Rosetta 2 para executar o Unity em dispositivos Apple Silicon.


Requisitos para o Unreal Engine

Requisitos mínimos:
- SO: Windows 7/8 64 bits ou superior
- Processador: 2,0+ GHz
- RAM: 2 GB de RAM no sistema
- Vídeo: DirectX10, junho de 2010
Requisitos para criação de conteúdo:
- SO: Windows 7/8 de 64 bits
- Processador: Quad-core Intel ou AMD, 2,5 GHz ou mais rápido
- RAM: 8 GB de RAM no sistema
- Vídeo: NVIDIA GeForce 470 GTX ou AMD Radeon 6850 HD ou superior
Requisitos para desenvolvimento com o Unreal Engine:
- Windows – Visual Studio 2013 Professional ou Visual Studio 2013 Express
- iOS – iTunes 11 ou superior / Mac OS X 10.9.2 Xcode 5.0 ou superior


Requisitos mínimos: Especificações básicas para executar a Unreal Engine.
Requisitos para criação de conteúdo: Especificações mais altas para usuários que criam conteúdo de jogo complexo.
Requisitos de desenvolvimento: Requisitos específicos de hardware e software para desenvolvedores que trabalham com a Unreal Engine, incluindo desenvolvimento para Windows e iOS.


Requisitos para o Flutter

- Para utilizar o Flutter: macOS de 64 bits e 700 MB de espaço em disco
- Espaço em disco total para o ambiente completo: 83 GB (pode variar)
- Itens obrigatórios incluem Adobe Photoshop, Apple iMovies, Apple Garage Band, etc.
- VMs/Simuladores: Google Pixel 2XL - Android VM, iPhone XR - iOS Simulator


Requisitos mínimos: Requisitos básicos para executar o Flutter em um sistema MacOS de 64 bits com 700 MB de espaço em disco.
Espaço total em disco: 83 GB são recomendados para o ambiente completo de desenvolvimento, incluindo várias ferramentas e dependências.
Itens adicionais: Certos softwares, como Adobe Photoshop, Apple iMovies e Apple Garage Band, são mencionados como obrigatórios, provavelmente para desenvolvimento gráfico e multimídia.
VMs/Simuladores: Dispositivos virtuais específicos (Google Pixel 2XL, iPhone XR) são recomendados para testar aplicativos Flutter em plataformas Android e iOS.
